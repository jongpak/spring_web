# 스프링 웹 MVC 기초 뼈대
새로운 스프링 웹  MVC 프로젝트를 위한 기초 뼈대입니다.

## 버전
* Java 1.8
* Spring framework 4.3.6
* Tomcat 7.0.75
* Tiles 3.0.7

## 구성
* [x] 초간단 HomeController
* [x] Tiles 레이아웃 구성
* [ ] JdbcTemplate
* [ ] Hibernate
* [ ] 개발/운영 Maven 프로필

## 이클립스 프로젝트 시작하기
`mvn eclipse:clean eclipse:eclipse` 명령을 실행하면 이클립스 프로젝트가 생성됩니다.
